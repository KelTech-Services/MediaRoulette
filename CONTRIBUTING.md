# Contributing to MediaRoulette

Thanks for your interest in MediaRoulette! This is a one-person project maintained in my free time, so I've kept the contribution process simple.

## How to Contribute

### Report Bugs & Request Features

The best way to contribute is through [GitHub Issues](https://github.com/KelTech-Services/MediaRoulette/issues/new/choose).

**For Bug Reports, please include:**

- A quick summary of the problem
- Steps to reproduce (be specific!)
- Your deployment method (Docker, Unraid, Portainer, etc.)
- MediaRoulette version (container tag)
- What you expected vs. what actually happened
- Logs or error messages (`docker logs <container_name>`)
- Anything you've already tried

**For Feature Requests:**

- Describe the feature and the problem it solves
- Include screenshots or mockups if helpful

### Join the Community

For general questions or discussion, join our [Discord server](https://discord.gg/hHvgWNnegq).
